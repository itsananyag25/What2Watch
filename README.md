# 🎬 What2Watch – AI/ML Movie Recommendation System

## 📌 Overview

What2Watch is an AI/ML-powered movie recommendation system built using Python and the TMDb API. It suggests personalized movie recommendations based on user input through a simple command-line interface (CLI).

---

## 🚀 Features

* 🎥 Movie recommendations based on user input
* 🌐 Real-time data using TMDb API
* 💻 Interactive command-line interface
* 🧠 AI/ML-inspired recommendation logic

---

## 🛠️ Tech Stack

* Python
* TMDb API
* Requests Library

---

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone <your-repo-link>
   ```

2. Navigate to the project folder:

   ```
   cd <your-project-folder>
   ```

3. Install required library:

   ```
   pip install requests
   ```

4. Run the program:

   ```
   python main.py
   ```

---

## 💻 How to Use

* After running the program, you will see:

  ```
  🎬 Movie Recommendation System
  1. Get Recommendations
  2. Exit
  ```

* Enter **1** to get recommendations

* Type a movie name (e.g., *Avengers*)

* View recommended movies

* Enter **2** to exit the program
