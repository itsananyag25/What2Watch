# Project Statement

## Project Title:
Movie Recommendation System Using TMDB API

## Course:
Fundamentals in AI and ML (CSA2001)

## Description:
This project presents an AI/ML-based Movie Recommendation System that provides real-time movie suggestions based on user input. The system integrates with the TMDB (The Movie Database) API to fetch movie details and recommended titles dynamically.

The application allows users to:
- Enter a movie name
- Retrieve the corresponding movie ID using TMDB Search API
- Fetch recommended movies using TMDB Recommendation API
- Display the top 5 recommended movies via a CLI interface

## Technologies Used:
- Python
- TMDB REST API
- Requests Library
- JSON Data Handling
- Command Line Interface (CLI)

## Purpose:
The objective of this project is to demonstrate practical implementation of API integration, real-time data retrieval, and intelligent recommendation systems within an AI/ML framework.

## Declaration:
This project is developed as part of the academic requirements for the course "Fundamentals in AI and ML." The implementation and documentation are prepared for educational purposes.
